# Agent Worklog — Session Activity Log & Handoff Notes

This file records agent activities, decisions made during sessions, context summaries, and handoff notes for continuity between OpenCode sessions.

## How to Use

When transitioning between sessions or delegating work to subagents, log key information here so the next agent can pick up without re-reading entire conversation history.

### Entry Format

```markdown
### [YYYY-MM-DD HH:MM] Session ID: ses_XXXXXX
**Agent**: [agent name/category]
**Task**: [brief description]
**Decisions Made**: [key architectural or implementation decisions]
**Next Steps**: [what the next agent should do]
**Blockers**: [any unresolved issues]
```
